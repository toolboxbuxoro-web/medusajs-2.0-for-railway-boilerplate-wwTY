# Быстрый старт: Тестирование модуля отзывов

## 1. Запуск миграций (ОБЯЗАТЕЛЬНО)

```bash
cd backend
npm run build
medusa migrations run
```

## 2. Перезапуск сервера

```bash
cd backend
npm run dev
```

## 3. Быстрый тест через curl

### Проверить, что API работает:
```bash
# Замените PRODUCT_ID на реальный ID товара
curl http://localhost:9000/store/products/PRODUCT_ID/reviews
```

Должен вернуть JSON с пустым массивом отзывов.

## 4. Тест через браузер

1. Откройте страницу товара: `http://localhost:8000/products/[product-id]`
2. Прокрутите до секции отзывов
3. Если вы авторизованы и получили товар - должна быть форма для отзыва

## 5. Проверка в базе данных

```bash
psql $DATABASE_URL

# Проверить таблицу
SELECT * FROM review LIMIT 5;

# Проверить индексы
\di review*
```

## Если что-то не работает

1. **Проверьте логи сервера** - там будут ошибки
2. **Проверьте миграции** - убедитесь, что они выполнены
3. **Проверьте конфигурацию** - модуль должен быть в `medusa-config.js`

См. `TESTING_REVIEWS_MODULE.md` для детального тестирования.
